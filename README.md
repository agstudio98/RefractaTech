# RefractaTech

Cadena de sucursales de productos tecnológicos y servicios especializados.

## Características
- Diseño Moderno estilo iOS Eficiente.
- Fondo 3D interactivo con Three.js.
- SPA (Single Page Application) fluida.
- Catálogo con filtrado por categorías.
- Soporte vía Chatbot interactivo.
- Formulario de contacto integrado.

## Tecnologías
- React 19
- TypeScript
- Vite
- Three.js / React Three Fiber
- CSS (Glassmorphism)

## Despliegue
El proyecto está configurado para desplegarse automáticamente en GitHub Pages.

```bash
pnpm run deploy
```
